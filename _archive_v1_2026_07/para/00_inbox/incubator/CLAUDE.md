# Incubator

Ideas not ready to become active projects yet. Each idea lives in a date-prefixed folder with its own `CLAUDE.md`.

## Lifecycle

1. **Capture**: `/capture-idea` creates a new idea folder here
2. **Accumulate**: `/add-context` adds links, docs, notes over time
3. **Promote**: `/start-project` moves an idea to `01_projects/` when ready
4. **Archive**: `/archive-project` removes ideas that are no longer relevant
